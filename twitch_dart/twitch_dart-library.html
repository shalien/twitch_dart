twitch_dart

twitch_dart library Null safety

A Twitch API wrapper in Dart. Allow to simply and easily use Twitch API for multiple client. Can be used in dart:io and dart:html.

Classes

TwitchClient
Allow to connect to the twitch api using a clientId and a token. The token can be a appAccessToken or a userAccessToken or a OAuthAccessToken.
